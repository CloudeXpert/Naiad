UtilsWriteByTimeTOutput, TClient, TWriter, TTime Method Naiad Help

[This is preliminary documentation and is subject to change.]

serialize a sequence of records to a collection of files partitioned by process, thread and time. For each process/thread/time this writes a sequence of files; each time a file reaches a threshold number of bytes, it is closed and another is opened. This keeps individual files of bounded length, allowing for more parallelism when reading them later
