The first version of Naiad, version 0.1 included an implementation of differential dataflow that ran on one or many computers.

Version 0.1 was released on October 8th 2012.

Changes in This Release

  • Serial, multicore, and distributed implementations of differential dataflow.

  • Example programs that illustrate how to use differential dataflow for a variety of graph and non-graph computations.
