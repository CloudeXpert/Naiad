Naiad is an open-source data-parallel dataflow system designed to provide low latency and high throughput for a variety of sophisticated data processing tasks.

Naiad exposes low-level primitives for dataflow construction and execution, as well as several libraries wrapping standard patterns in higher level constructs. Our experience has been that many popular programming patterns are easily expressed in Naiad, including streaming analytics, graph computation, and machine learning. Our intent is that users will mostly use common high-level patterns, but have access to low-level primitives as needed, which can be wrapped and shared as new high-level patterns.

Getting Started

You will want a recent version of Naiad. Naiad is available both as binaries on NuGet and as open source on GitHub.

This document Writing your first Naiad program has an introduction to writing Naiad programs.

Learning more about Naiad

Naiad is an ongoing research project, but there are many places to find more information about what has already been done and what is coming next.

  • The Naiad project page has information about the research that underlies the Naiad system. It also discusses research related to Naiad, and has links to other Microsoft Research projects related to large-scale data processing.
  • The Big Data at SVC blog contains many posts about Naiad, related projects, and opinions about the state of large-scale data processing generally.
  • Naiad's GitHub project page is a great resource for the source code itself, as well as a place to comment on and discuss the current state of the code and its future directions.
