Version 0.2Naiad Help

[This is preliminary documentation and is subject to change.]

The first open-source release of Naiad, version 0.2 included support for multiple data-parallel frameworks on top of a common low-latency and high-throughput distributed runtime.

Version 0.2 was released on October 29th 2013.

Changes in This Release
  • The license was changed to the Apache 2.0 open-source license.
  • The Naiad runtime was split into the Naiad.dll runtime, and multiple frameworks. This release contained the Lindi (LINQ with declarative iteration) and Differential Dataflow frameworks.
